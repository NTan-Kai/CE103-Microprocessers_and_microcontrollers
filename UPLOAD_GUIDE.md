# Upload Guide

Copy all folders in this package into the root of your CE103 GitHub repository, then commit and push.

```bash
git clone https://github.com/NTan-Kai/CE103-Microprocessors_and_microcontrollers.git
cd CE103-Microprocessors_and_microcontrollers
# copy package contents here
git add .
git commit -m "Add CE103 labs and final project"
git push
```
