# Smart Door Lock System Using 8051, RFID and SD Card

This final project implements a smart door lock system using the AT89S52 microcontroller, RFID module, LCD, servo motor, and SD card storage.

## Contents

- `code/`: Keil and Proteus project files, source code, and generated HEX file.
- `report/`: Final project report.
- `presentation/`: Final project presentation slides.

## Main Features

- RFID card reading through UART.
- UID checking and storage on SD card.
- LCD display through I2C/PCF8574.
- Servo-based lock control.
- Admin mode for adding new cards.
